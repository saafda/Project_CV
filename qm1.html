One dimensional Schrödinger equation

Numerical solution of the schrödinger equation in one dimension with different potentials

In this project the goal was solve the schrödinger equation in one dimension using the diagonal hamiltonian matrix. Applying the hamiltonian matrix on the wave function in vector space results in an eigenvalue vector with diagonal energy values. Additionally the eigenfunction will then be the numerical values for the corresponding wave function.
