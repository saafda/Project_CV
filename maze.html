Maze Solver Application

Maze solver algorithm used as solver in maze game

This project is an app that I developed in an Object-oriented programming course i took in my second semester. My goal for the project was to use a recursive algorithm to solve a randomly generated maze. I then used this as the idea for a maze solving game where the app randomly generates (prim's algorithm) a maze then the app or the player can solve the maze.
