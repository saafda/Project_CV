Time dependant Schrödinger equation

Animating the time dependant wave function using different potentials

The goal of this project was to use the solving algroithm from my other project concerning the Schrödinger equation in one dimension to find and animate the wave function using matplotlib. This is possible thanks to the the fact that the time dependant wave function can be described by the product of the sationary state and the time-evolution operator.
